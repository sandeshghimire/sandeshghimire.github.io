Xilinx Zynq 7000 (ZedBoard)


Xilinx's all programmable SoCs are processor-centric platforms that offer software, hardware and I/O programmability in a single chip. The Zynq-7000 family is based on the All Programmable SoC architecture. Zynq-7000 products incorporate a dual core ARM Cortex-A9 based Processing System (PS) and Xilinx Programmable Logic (PL) in a single device.

ZedBoard is a complete development kit for designers interested in exploring designs using the Xilinx Zynq-7000 All Programmable SoC. . This section describes a group of projects which help to jumpstart the development process. The project include bare metal implementation with FreeRTOS , Linux Root File System (RFS), Linux Software Development Kit (LSDK) and a final project with both FreeRTOS and Linux running side by side on the same machine

FreeRTOS Port for ZedBoard

FreeRTOS is an industry standard open source real time operating system. It is popular and runs in wide variety of microcontrollers from Cortex M0/M3/M4 through A7/A9 SoC. FreeRTOS is versatile and if you are looking for a real time performance from your Zynq-7000 SoC it is the right solution.

In this example i have created 3 threads and a software timer, the tx task, rx task and a dma task. These task have unique priority and defined stack sizes. In addition there is a software timer which fires periodically at 10 Hz. here

ZedBoard SD Card Image

The SD card image contains all the essential components to boot the linux operating system. The image has two partition, the first partition has two paration, the first one is FA32 paration which contains the First Stage Bootloader (FSBL). The bootloader (U-BOOT). The device tree (TD) and Linux Kernel .

The second partition of the SD card contains the Root File System (RFS) based on linux standard base (LSB). The RFS contains all the essentials file and library to boot the ZedBoard. This RFS is developed from scratch for the ZedBoard therefore it is not part of any distribution but ZedBoard and every component has been configured and optimized for ZedBoard. here
